## 2. 环境配置
### 1. 使用 Poetry(建议)
```sh
poetry env use "path to your python executable"
poetry install
```


### 2. 使用 Conda
```sh
conda create -n mediaaio python=3.10
conda activate mediaaio
pip install -r requirements.txt
```

## 3. 项目使用

请确保在使用前完成相关的环境配置。

```sh
git clone --recursive https://github.com/NightWatcher314/MediaAIO
cd MediaAIO

poetry shell
or 
conda activate mediaaio

gradio webui.py
```
